# Triangulo

Enunciado

```
Ingresar por teclado 3 (tres) números correspondientes a los lados de un triángulo.
Teniendo en cuenta que la suma de los dos lados menores tiene que ser superior al lado mayor para poder formar un triángulo.
Indicar si los números ingresados forman un triángulo y qué tipo de triángulo es según sus lados (equilátero, isósceles o escaleno).
```

Resolucion

```
La solucion del ejercicio se compone de tres componentes:
- Proyecto de biblioteca: representa la clase triangulo
- Proyecto de consola: representa la interfaz de usuario del programa
- Proyecto de test unitario: representa pruebas con casos posibles a cada método realizado
```
